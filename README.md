# ВолонтёрМДК — Локальный сервер (Python)

## Запуск — 1 команда, ничего устанавливать не нужно!

### Windows
Дважды кликни `ЗАПУСТИТЬ.bat`
или открой папку в терминале и введи:
```
python server.py
```

### Mac / Linux
```
python3 server.py
```

---

## Открой в браузере
```
http://localhost:3000
```

Можно открыть в **нескольких вкладках** одновременно с разными аккаунтами.
Все изменения сразу видны во всех вкладках (WebSocket реалтайм).

## Как тестировать с разных аккаунтов

1. Открой `http://localhost:3000` в **обычной** вкладке → войди как `admin@demo.kz`
2. Открой `http://localhost:3000` в **режиме инкогнито** → войди как `volunteer@demo.kz`
3. Действия в одной вкладке сразу видны в другой!
